# User Guide

Admin opens `/admin`, logs in with the admin PIN, and manages the committee from tabs:

- Dashboard: live overview.
- PN: create, edit, introduce, reject, start debate, or start voting.
- Voting: close, reopen, save, cancel, and force projection update.
- Speakers: add delegations to queue, add reactions, move to next speaker.
- Attendance: check in delegations and generate delegate voting codes.
- Layout: adjust projection seat flags.
- Breaks: start/end caucus or break overlays.
- Agenda: admin-only program management.
- Settings: conference labels, time limits, PIN changes, resets.

Projection opens `/screen` and logs in with the screen PIN. It shows only clock/layout, resolution, speakers, and overlays. Agenda and participant personal data are not sent to projection.

Delegates open `/vote`, enter a four-digit attendance code, vote with large buttons, view the current resolution, and submit amendments. The delegate UI never shows participant names, emails, co-delegate data, or notes.
